# vja-learn-net

A project by **Victor Jeman** and **Taia Dimitrova** to learn **C#** and **.NET**,
with a strong focus on writing tests.

We work through [Exercism C#](https://exercism.org/tracks/csharp/exercises)
exercises, but treat each one as a *feature* folded into a single growing app, not
as a throwaway puzzle.

## Commands

Requires **.NET SDK 10** (pinned in `global.json`).

```bash
dotnet run --project src/VjaLearnNet         # run the app
dotnet watch run --project src/VjaLearnNet   # run with hot reload (≈ pnpm dev)
dotnet test                                  # run all tests
```
